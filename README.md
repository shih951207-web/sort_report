## 排序法介紹
### 氣泡排序法 (Bubble Sort)
透過相鄰元素不斷兩兩比較並交換，讓最大的元素像氣泡一樣逐漸浮到數列末端

### 選擇排序法 (Selection Sort)
每一輪從未排序區中找出最小的元素，直接與未排序區的第一個位置交換

### 插入排序法 (Insertion Sort)
將數列分為已排序與未排序兩部分，逐一將未排序元素插入到已排序序列中的正確位置

### 合併排序法 (Merge Sort)
採用分治法，將數列不斷對半分直到最小單位，再依序合併回已排序的大數列

### 堆積排序法
選定一個基準點，將數列分成比它小與比它大的兩部分，再遞迴處理這兩部分
